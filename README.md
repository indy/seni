




prerequisite:

$ http-server

## Build

### Prerequisites:

1. `npm install`

### Build:

1. `gulp build`

### Tests:

1. `gulp test`

### Dev

1. `gulp build-watch`: Watches src folder for changes
2. `gulp serve`
3. open http://localhost:8000
