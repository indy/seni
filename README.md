
## Build

### Prerequisites:

1. `npm install`
2. `npm install -g gulp bower`
3. `bower update`
4. `gulp build`


### Folder structure

### Build:

1. `gulp build` -> result is in `dist` folder
2. `gulp clean` -> cleans the `dist` folder

### Tests:

1. `gulp test`
